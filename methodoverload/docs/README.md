# PyOverload

PyOverload is a Python library that allows **method and function overloading** using type hints.  
Supports **free functions**, **instance methods**, **static methods**, and **class methods**.

---

## Features

- Overload functions based on argument types.
- Supports instance, class, and static methods.
- Automatic overload resolution using type hints.
- Caching for faster repeated calls.
- Metaclass support for class-based overloads.
